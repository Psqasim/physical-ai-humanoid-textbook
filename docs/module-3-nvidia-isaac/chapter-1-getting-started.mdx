# Chapter 1: Getting Started with NVIDIA Isaac

Learn the basics of NVIDIA Isaac Sim and how to run your first AI-powered robot simulation.

## Topics Covered

- Installing Isaac Sim (Omniverse platform)
- Understanding the Isaac Sim interface
- Loading pre-built robot models (Carter, Jetbot)
- Running perception models for object detection

## Hands-On Exercise

By the end of this chapter, you'll run an object detection model on a simulated robot camera feed in Isaac Sim.
