# ROS 2: ロボットの神経系

このモジュールでは、次世代のロボティクス用ミドルウェアであるROS 2（Robot Operating System 2）を紹介します。node、topic、service、actionなどの中核概念と、ROS 2が分散ロボットシステムをどのように実現するかを学びます。

## 学習内容

- ROS 2のアーキテクチャと設計思想
- nodeの作成と通信管理
- デバッグのためのrqtやrvizなどのツールの使用
- ROS 2とシミュレータ及び実機ハードウェアの統合

## 前提知識

- 基本的なプログラミングスキル（PythonまたはC++）
- Linuxコマンドラインの基礎知識
- ロボティクスの基本概念の理解（必須ではありませんが有用です）
