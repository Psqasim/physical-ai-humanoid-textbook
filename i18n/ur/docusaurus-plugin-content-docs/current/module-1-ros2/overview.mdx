# ROS 2: روبوٹ کا اعصابی نظام

یہ ماڈیول ROS 2 (Robot Operating System 2) متعارف کراتا ہے، جو روبوٹکس کے لیے اگلی نسل کا middleware ہے۔ آپ بنیادی تصورات جیسے node، topic، service، اور action سیکھیں گے، اور یہ کہ ROS 2 کس طرح تقسیم شدہ روبوٹ سسٹمز کو فعال کرتا ہے۔

## آپ کیا سیکھیں گے

- ROS 2 کی تعمیرات اور ڈیزائن فلسفہ
- Node بنانا اور رابطے کا انتظام
- ڈیبگنگ کے لیے rqt اور rviz جیسے ٹولز کا استعمال
- ROS 2 کو simulators اور حقیقی hardware کے ساتھ ضم کرنا

## پیشگی ضروریات

- بنیادی پروگرامنگ مہارتیں (Python یا C++)
- Linux command line سے واقفیت
- بنیادی روبوٹکس تصورات کی سمجھ (اختیاری لیکن مفید)
